# visual

Навык морды для визуальной вёрстки поста через связку «сервер + Pencil». Сервер даёт бриф и фирстиль (`get_visual_brief`, `get_brand_identity`/`set_brand_identity`) и проверяет готовый дизайн (`validate_visual_design`); рендерит `.pen` сам Pencil MCP на стороне морды.

Разделение труда: сервер = бриф + бренд + бренд-QA, Pencil = рендер. Сервер не сочиняет текст (agent-as-author) - он только размещает переданный `copy` по зонам и детерминированно проверяет результат (контраст, палитра, шрифты, поля, логотип). См. `SKILL.md`.

Карусели и брендовые баннеры собираются ТОЛЬКО серверными тулзами (`create_carousel_from_idea` или `get_visual_brief` -> Pencil -> `validate_visual_design`). Самописный рендер (PIL, python, matplotlib, canvas) с захардкоженными цветами запрещён: `create_carousel_from_idea` сам берёт палитру, шрифты и логотип из бренд-профиля на сервере.
