# copilot

Головной навык-оркестратор контент-завода для лидогенерации. Связывает весь конвейер в одну линию: разобрать бизнес (`get_my_account`, при пробелах -> навык `onboarding`) -> спланировать по лестнице Ханта (`list_hunt_stages`, `get_content_plan`, `select_content_framework`/`list_content_frameworks`) -> опционально собрать референсы через Apify (`ingest_competitor_content`, `ingest_reviews_or_site`, `list_ingested_references`, помечены `untrusted_external_data: true`) -> произвести контент (тексты через `score_content_text`/`refine_content_text` плюс GEO `score_citability`/`score_answer_structure`, карусели `create_carousel_from_idea`, видео `create_flexi_video`/`list_flexi_modes`, статьи `get_article_outline`->`validate_article`/`score_article`->`register_article`->`suggest_internal_links`/`check_cannibalization`, картинки `generate_image`, брендовый визуал через навык `visual`) -> ревью и публикация (`create_content_unit`/`review_content`, `create_publish_batch`/`approve_publish_batch`, `get_integration_status`) -> петля аналитики (`fetch_post_metrics`).

Принцип agent-as-author: тексты пишет морда, сервер только валидирует, оценивает, рендерит, хранит и публикует. Рамка - бизнес-инструмент лидогенерации, не SMM ради лайков. Платные действия метрятся против лимита расходов клиента (`get_my_account` -> `remaining_cents`). См. `SKILL.md`.
